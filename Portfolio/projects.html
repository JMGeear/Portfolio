A Programmer's Portfolio

The Viraltone Website:

Coded in HTML5, this fully responsive site is allows Viraltone fans to view biographies of each member, check schedules and listen to songs performed by the band.

Content Management System:

Coded in PHP and using MySQL this content management system allows customers to create dynamic pages stored in a database to be displayed to users on the front end. The site also allows administrators to manage users and administrators stored in the back end databases.

Git Hub repository:

This contains the code for my projects including this portfolio site. The Portfolio site is a fully responsive website uses Foundation as the framework and Slick as the media slider.
